# Perfume Matching Application

Welcome to the Perfume Matching Application! This application allows users to find perfume alternatives based on their favorite scents. Users can input the name of their favorite perfume, and the application will suggest similar perfumes from the database.

## Setup

Before you get started, make sure you have the following prerequisites:

- Python
- Flask
- MySQL (or another database of your choice)

You can install Flask and other dependencies using pip:

```bash
pip install Flask flask_sqlalchemy mysql-connector-python
